# Customer Shopping Behavior Analysis

## Overview

This project analyzes retail customer shopping behavior to uncover purchasing trends, customer preferences, and factors influencing sales. Using Python, MySQL, and Power BI, the project transforms raw data into actionable business insights through data cleaning, SQL analysis, and interactive dashboards.

## Business Problem

A retail company wants to understand customer shopping behavior to improve customer engagement, optimize marketing strategies, and increase revenue. This project identifies purchasing patterns across demographics, product categories, discounts, reviews, payment methods, and subscription status.

## Objectives

- Clean and preprocess raw customer data using Python.
- Store and analyze data using MySQL.
- Answer business questions through SQL queries.
- Build an interactive Power BI dashboard.
- Provide business recommendations based on insights.

## Key Business Questions

- Revenue generated by male vs female customers

- Shipping preference analysis
- Seasonal purchasing trends
- Product review performance
